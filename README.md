# Personal Website Portfolio

This is a personal website portfolio that includes a homepage, an about me section, my image, and social media icons.

## Features

- **Home**: Introduction and welcome message.
- **About Me**: A section describing my background, skills, and experience.
- **Image**: A personal image of myself.
- **Social Media**: Links to my social media profiles (LinkedIn, GitHub, etc.).

## Technologies Used

- HTML
- CSS
- Box Icons (for social media icons)
